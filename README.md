# ThreadForge AI 🚀

ThreadForge AI is an AI agent that helps generate viral Twitter threads, content ideas, and short-form scripts instantly.

---

## ✨ What it does

- Generates Twitter threads  
- Creates viral content ideas  
- Writes short-form scripts  

---

## 🚀 How to use

1. Open the app  
2. Select ThreadForge AI  
3. Click New Chat  
4. Type:
   Write a viral crypto thread about AI agents  

---

## 💡 Example Prompt

Write a viral crypto thread about AI agents

---

## ⚙️ Deployment

Deployed on Nosana with GPU and accessible via browser.
